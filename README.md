# Soft LED pulses in a pleated Danish Christmas Heart
<div align="center">
  <img src="./output-opt.gif" alt="Demo of feature">
</div>
An Arduino controls a modified multicoloured 40-LED bulb chain (reduced to 20 LEDs) housed inside a pleated Danish julehjerte, creating a calm, heartbeat-like light pulse.

## Materials 
- Arduino Uno
- A pleated christmas heart of standard size
- LED light chain (for more see [LED_chain_specs.json](https://github.com/maxodsbjerg/beating-LED-julehjerte/blob/main/LED_chain_specs.json)
  - LED chain current (measured with meter)
    180 mA
  - Total power consumption ~0.58 W
- MOSFET BUK455500B
- 220 Ω, ±5% - resistor
